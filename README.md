# 활성탄 경도 (Carbon Hardness)

활성탄 경도측정 실험을 위한 핸즈프리 안드로이드 앱.
실험 중 손이 더러워 폰을 만질 수 없는 상황을 위해 **음성 인식**으로 조작하고,
타이머는 **백그라운드에서 진동 알람**으로 알려줍니다.

## 주요 기능

- 🎤 **한국어 음성 무게 입력** — "일번 오십쩜사사" → 1번 칸에 50.44 g 기록, "응/아니"로 확정
- ⏱️ **3개 공정 타이머** — 거르기 10분 · 쇠구슬(마모) 30분 · 체진동기(미세체) 3분
  - AlarmManager 기반: 앱을 꺼도, 다른 앱을 써도, 절전 중에도 정확히 울림
  - 완료 **예정 시각**(예: 14:42)을 화면·알림에 표시
- 📳 **진동 전용 알람** — 소리 없이 확인할 때까지 반복 진동 + 전체화면 알림
- ✅ **질량 검증** — (가벼운+무거운) vs 초기무게 오차 2% 이내 자동 판정
- 💾 **상태 저장** — 앱을 나갔다 돌아와도 진행 상황 그대로 복원

## 실험 흐름

| 단계 | 작업 | 입력 |
|---|---|---|
| 1 | 채 2개로 거르기 (10분) | "거르기 시작" |
| 2 | 초기무게 M₀ 측정 (10 mL 부피) | "일번 오십쩜사사" |
| 3 | 쇠구슬 넣고 마모 (30분) | "마모 시작" |
| 4 | 체진동기로 분리 (3분) | "시작" |
| 5 | 무거운 · 가벼운 부분 계량 | "이번 …", "삼번 …" |

경도(%) = 무거운 ÷ 초기무게 × 100

## 빌드

Android Studio 또는 CLI:

```bash
./gradlew assembleDebug
```

- minSdk 26 (Android 8.0+), targetSdk 34
- Kotlin + Jetpack Compose
- 음성인식: Android SpeechRecognizer (ko-KR, 오프라인 지원)
